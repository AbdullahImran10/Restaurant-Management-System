# Restaurant Management System (RMS)

A console-based Restaurant Management System developed in **C#** using **Object-Oriented Programming (OOP)** principles. The application helps manage customers, menu items, orders, reservations, employees, billing, and reports.

## Features

### Customer Management
- Add new customers
- Automatically generate Customer IDs
- Track customer visits
- Loyalty points system
- Redeem loyalty points for discounts

### Menu Management
- Add, update, and remove menu items
- Categorize menu items (Starter, Main Course, Dessert)
- Automatic Item ID generation
- View complete menu

### Order Management
- Create new orders
- Add multiple menu items to an order
- Track order status:
  - Pending
  - Preparing
  - Ready
  - Completed
  - Cancelled
- Calculate order total

### Billing System
- Generate customer bills
- Apply loyalty point discounts
- Display final payable amount

### Reservation Management
- Reserve restaurant tables
- View reservation details
- Manage table availability

### Employee Management
- Store employee information
- Manage employee records

### Reporting
- View customer information
- Display menu details
- View orders
- Generate restaurant reports

## How to Run

### Prerequisites

- .NET SDK installed
- Visual Studio or Visual Studio Code

### Steps

1. Clone the repository

```bash
git clone https://github.com/your-username/RestaurantManagementSystem.git
```

2. Navigate to the project directory

```bash
cd RestaurantManagementSystem
```

3. Run the application

```bash
dotnet run
```
